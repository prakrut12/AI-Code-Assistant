#🤖 AI-Code-Assistant

🚀 Overview:

Agentic Coding Assistant is a Python-based application that simulates AI-assisted coding. It allows users to input coding problems and receive solutions, along with the ability to improve or refine them.
This project demonstrates the concept of **agentic coding**, where a system behaves like an intelligent assistant that can take input, make decisions, and generate solutions.



 💡 Features
- 🔹 Accepts user input for coding problems
- 🔹 Generates solutions for problems like:
  - Factorial
  - Fibonacci
  - Prime number
  - Sum of natural numbers
- 🔹 Allows improvement/debugging of generated code
- 🔹 Simple and interactive UI using Streamlit

🧠 What is Agentic Coding?
Agentic coding refers to using intelligent systems (AI agents) that can:
- Understand user input
- Decide actions
- Generate solutions
- Improve results iteratively

This project simulates that behavior using logic-based decision making and can be extended with real AI APIs.



 🛠️ Tech Stack
- Python
- Streamlit (for UI)

## ▶️ How to Run

1.Clone the repository:
   git clone https://github.com/your-username/agentic-coding-assistant.git

2.Navigate to project folder:
   cd agentic-coding-assistant

3.Install dependencies:
   pip install streamlit

4.Run the app:
   python -m streamlit run app.py
